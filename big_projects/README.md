# Quantum‑Inspired Projects Overview

This `big_projects` directory collects five experimental projects based on the Zero framework’s mathematical models.  Each folder contains a self‑contained Python module and demonstration code implementing one of the core equations.  The projects showcase how quantum‑inspired decision equations can be used to simulate adaptive behaviour, systemic learning, key generation and cognitive optimisation.

## Project List

| Project | Description |
| --- | --- |
| **adaptive_decision_framework** | Implements the Adaptive Learning and Decision Equation (Z function) and demonstrates how to evaluate it over an array of inputs. |
| **genetic_adaptation_simulator** | Implements the Genetic Adaptation Equation (G function) with a simple simulation over a range of values. |
| **quantum_key_solver** | Provides a solver for the Quantum Key Equation (F function), including support for piecewise behaviour through Heaviside approximations. |
| **cognitive_optimization_engine** | Implements a simplified Cognitive Optimization Equation (C function) that can be evaluated over a grid of `x` and `y` values. |
| **quantum_ai_examples** | Contains a small script demonstrating how to use the above models together. |

Each project README contains detailed usage instructions and describes the mathematical background.  This file serves as an entry point for users exploring the collection.
