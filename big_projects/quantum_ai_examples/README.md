# Quantum AI Examples

This repository provides a set of simple examples demonstrating how to use the models implemented in the other projects of this collection.  By combining the Adaptive Decision Framework, Genetic Adaptation Simulator, Quantum Key Solver and Cognitive Optimization Engine, these examples illustrate how quantum‑inspired mathematics can be applied to real‑world problems.

## Contents

- `example_usage.py` – runs sample calculations using the functions defined in the sibling projects.
- `README.md` – documentation and guidance.

## Running the Example

Before running the script, ensure that the modules from the other projects are available on your Python path.  One way to achieve this is to place all the `big_projects` directories into a single package or install them as editable packages.  For the purposes of this demonstration, you can modify `PYTHONPATH` or copy the modules into the same directory.

To run the example:

```bash
python example_usage.py
```

The script will compute sample values for each model using default parameters and print a summary of the results.
