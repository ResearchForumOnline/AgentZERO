TACTICAL OPERATIONS v4.1

01 // BARE METAL INITIALIZATION

Standard deployment for local Linux environments (AlmaLinux / Ubuntu / Debian).

cd ~/openzero && ./start_openzero_ui.sh

Mission Control: Access via http://localhost:1024

02 // DOCKER CONTAINMENT

For isolated deployment or cloud clusters. Requires Docker Engine 24.0+.

Build the Image:

docker build -t openzero-node .

Run the Container:

docker run -d -p 1024:1024 --name zero_node_1 openzero-node

Note: To persist RAG memory, mount a volume to /app/chroma_db.

04 // HARDWARE REQUIREMENTS

Minimum Specs for Gemma 2 (9B):

  • CPU: 8 Cores (AVX2 Support required)
  • RAM: 16GB System Memory (Model uses ~8GB)
  • Storage: 20GB SSD (For Vector Index + Model Weights)
  • GPU: Optional. Vulkan/CUDA supported for faster inference.
